<p><b>Klaudyna Potocka</b>, z domu <a href="/wiki/Dzia%C5%82y%C5%84scy_herbu_Ogo%C5%84czyk" title="Działyńscy herbu Ogończyk">Działyńska</a> (ur. <a href="/wiki/27_sierpnia" title="27 sierpnia">27 sierpnia</a> <a href="/wiki/1801" title="1801">1801</a> na zamku w <a href="/wiki/K%C3%B3rnik" title="Kórnik">Kórniku</a>, zm. <a href="/wiki/8_czerwca" title="8 czerwca">8 czerwca</a> <a href="/wiki/1836" title="1836">1836</a> w <a href="/wiki/Genewa" title="Genewa">Genewie</a>) – polska szlachcianka, patriotka, zasłużyła się m.in. pomocą rannym w czasie powstania listopadowego  
</p>
<div class="mw-heading mw-heading2"><h2 id="Życiorys"><span id=".C5.BByciorys"></span>Życiorys</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Klaudyna_Potocka&amp;veaction=edit&amp;section=1" title="Edytuj sekcję: Życiorys" class="mw-editsection-visualeditor"><span>edytuj</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Klaudyna_Potocka&amp;action=edit&amp;section=1" title="Edytuj kod źródłowy sekcji: Życiorys"><span>edytuj kod</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Córka <a href="/wiki/Ksawery_Szymon_Dzia%C5%82y%C5%84ski" title="Ksawery Szymon Działyński">Ksawerego Działyńskiego</a> (senatora-wojewody <a href="/wiki/Ksi%C4%99stwo_Warszawskie" title="Księstwo Warszawskie">Księstwa Warszawskiego</a> i <a href="/wiki/Kr%C3%B3lestwo_Polskie_(kongresowe)" title="Królestwo Polskie (kongresowe)">Królestwa Kongresowego</a>), siostra <a href="/wiki/Tytus_Dzia%C5%82y%C5%84ski" title="Tytus Działyński">Tytusa Działyńskiego</a>, żona <a href="/wiki/Andrzej_Bernard_Potocki" title="Andrzej Bernard Potocki">Bernarda Potockiego</a> – syna pisarza <a href="/wiki/Jan_Potocki_(1761%E2%80%931815)" title="Jan Potocki (1761–1815)">Jana Potockiego</a>, autora <i><a href="/wiki/R%C4%99kopis_znaleziony_w_Saragossie_(powie%C5%9B%C4%87)" title="Rękopis znaleziony w Saragossie (powieść)">Rękopisu znalezionego w Saragossie</a></i>, przyjaciółka <a href="/wiki/Emilia_Sczaniecka" title="Emilia Sczaniecka">Emilii Sczanieckiej</a>.
</p><p>Pochodziła z zasłużonej dla kraju rodziny <a href="/wiki/Dzia%C5%82y%C5%84scy_herbu_Ogo%C5%84czyk" title="Działyńscy herbu Ogończyk">Działyńskich</a>. Jej ojciec Ksawery Szymon Działyński, razem z bratem <a href="/wiki/Ignacy_J%C3%B3zef_Dzia%C5%82y%C5%84ski" title="Ignacy Józef Działyński">Ignacym</a>, należał do aktywnych uczestników prac <a href="/wiki/Sejm_Czteroletni" title="Sejm Czteroletni">Sejmu Czteroletniego</a> i twórców <a href="/wiki/Konstytucja_3_maja" title="Konstytucja 3 maja">Konstytucji 3 maja</a>. Po śmierci ojca, matka Klaudyny odbyła z nią podróż do <a href="/wiki/Warszawa" title="Warszawa">Warszawy</a>, <a href="/wiki/Wiede%C5%84" title="Wiedeń">Wiednia</a> i <a href="/wiki/Pary%C5%BC" title="Paryż">Paryża</a>. Tam Klaudyna pobierała lekcje gry na fortepianie, dykcji, tańca, rysunków u znanych mistrzów. W <a href="/wiki/1823" title="1823">1823</a> roku powróciła z matką do <a href="/wiki/Konarzewo_(powiat_pozna%C5%84ski)" title="Konarzewo (powiat poznański)">Konarzewa</a>, gdzie 25 września 1825 poślubiła Andrzeja Bernarda Potockiego. Zmarła 8 czerwca 1836 z powodu powikłań chorób płuc i serca, będąc w siódmym miesiącu ciąży<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>.
</p><p>Zasłużyła się swą pełną poświęcenia pracą nad pielęgnowaniem rannych w dobie <a href="/wiki/Powstanie_listopadowe" title="Powstanie listopadowe">powstania listopadowego</a>. Po jego upadku osiedliła się w <a href="/wiki/Drezno" title="Drezno">Dreźnie</a>, gdzie – spieniężywszy swe klejnoty – niosła pomoc <a href="/wiki/Emigrant" title="Emigrant">emigrantom</a>. Tam nawiązała bliski kontakt z <a href="/wiki/Adam_Mickiewicz" title="Adam Mickiewicz">Mickiewiczem</a>, <a href="/wiki/Wincenty_Pol" title="Wincenty Pol">Polem</a>, <a href="/wiki/Antoni_Edward_Odyniec" title="Antoni Edward Odyniec">Odyńcem</a>, <a href="/wiki/Stefan_Garczy%C5%84ski_(poeta)" title="Stefan Garczyński (poeta)">Garczyńskim</a>, którego pielęgnowała następnie w chorobie. Dzięki zapisanym przez nią funduszom powstała w <a href="/wiki/Pary%C5%BC" title="Paryż">Paryżu</a> instytucja dobroczynna „Czci i chleba” roztaczająca opiekę nad emigrantami.
</p><p>Właścicielka Wyspy Zamkowej na <a href="/wiki/Jezioro_G%C3%B3reckie" title="Jezioro Góreckie">Jeziorze Góreckim</a> wraz z romantycznym zamkiem. Utrzymywała Schronisko Polskie w <a href="/wiki/Drezno" title="Drezno">Dreźnie</a>, wspomagała prasę emigracyjną.
</p><p>Została pochowana na <a href="/wiki/Cmentarz_Les_Champeaux_w_Montmorency" title="Cmentarz Les Champeaux w Montmorency">cmentarzu w Montmorency</a> koło <a href="/wiki/Pary%C5%BC" title="Paryż">Paryża</a>.
</p><p>W 1984 została odznaczona pośmiertnie <a href="/wiki/Gwiazda_Wytrwa%C5%82o%C5%9Bci" title="Gwiazda Wytrwałości">Gwiazdą Wytrwałości</a><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> – odznaczeniem prywatnym nadawanym przez bliżej nieokreśloną kapitułę.
</p>
<div class="mw-heading mw-heading2"><h2 id="Upamiętnienie"><span id="Upami.C4.99tnienie"></span>Upamiętnienie</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Klaudyna_Potocka&amp;veaction=edit&amp;section=2" title="Edytuj sekcję: Upamiętnienie" class="mw-editsection-visualeditor"><span>edytuj</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Klaudyna_Potocka&amp;action=edit&amp;section=2" title="Edytuj kod źródłowy sekcji: Upamiętnienie"><span>edytuj kod</span></a><span class="mw-editsection-bracket">]</span></span></div>
<ul><li>We Lwowie zostało założone Towarzystwo imienia Klaudii z Działyńskich Potockiej<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup>, podczas <a href="/wiki/Powstanie_styczniowe" title="Powstanie styczniowe">powstania styczniowego</a> Komitet Dam Rewolucyjny tzw. „Komitet Sióstr Klaudii”, powszechnie określany jako „Klaudynki”<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup>.</li>
<li>Na samym początku historii skautingu w Polsce, w grudniu 1911&#160;r., powstała I Drużyna Skautek im. Klaudii z Działyńskich Potockiej w <a href="/wiki/Krak%C3%B3w" title="Kraków">Krakowie</a> pod komendą Romy Wodziczko-Dediowej, w oparciu o pierwszy zastęp żeński założony we wrześniu przez Zofię Langierównę.<sup id="cite_ref-Vademecum_6-0" class="reference"><a href="#cite_note-Vademecum-6">[6]</a></sup></li>
<li>Klaudyna Potocka została patronką <a href="/wiki/V_Liceum_Og%C3%B3lnokszta%C5%82c%C4%85ce_im._Klaudyny_Potockiej_w_Poznaniu" title="V Liceum Ogólnokształcące im. Klaudyny Potockiej w Poznaniu">V Liceum Ogólnokształcącego w Poznaniu</a> oraz ulicy w tymże mieście.</li>
<li>W <a href="/wiki/Ko%C5%9Bci%C3%B3%C5%82_%C5%9Bw._Wojciecha_w_Poznaniu" title="Kościół św. Wojciecha w Poznaniu">kościele św. Wojciecha w Poznaniu</a> znajduje się tablica upamiętniająca Klaudynę Potocką.</li></ul>
